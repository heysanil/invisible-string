# Context & MCP

Context is the library your Agents are equipped from. It comes in two forms:
**MCP connections** — Model Context Protocol servers, at workspace or personal
scope — and authored **skills** with drag-and-drop attachments. Equipping an
Agent packages both straight into its compiled artifact, so a published Agent
carries its tools and knowledge with it.

The Context section includes an MCP registry browser and one-click install. Any
credentials a connection needs are stored write-once and encrypted; the API only
ever reports whether a connection has credentials, never their values.
