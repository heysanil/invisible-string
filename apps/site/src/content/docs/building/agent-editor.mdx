# The agent editor

The agent editor is where an Agent takes shape. A section rail — Persona,
Model, Context, Access — sits alongside focused editors for each, with the AI
copilot docked on the right. Edits autosave to the draft as you work, and
problems surface directly on the section that owns them.

The rail carries the Agent's lifecycle: **Chat with agent** opens a
conversation against the latest published version, and **Publish** compiles
and builds the draft with live build progress — an Agent is the thing that
builds, so publishing is where the draft becomes a real, version-pinned
artifact. The Access section sets the run-as user whose credentials the Agent
uses on every dispatch.
